# STM32 BME280 library via SPI

This repository provides a C library for the Bosch BME280 sensor. It is aimed at **STM32 microcontrollers** using the **SPI** bus.
* Adapted from the I<sup>2</sup>C BME280 library provided by Controllerstech (Source: [https://controllerstech.com/interface-bme280-with-stm32/](https://controllerstech.com/interface-bme280-with-stm32/)).
